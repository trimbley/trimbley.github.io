ဘရောင်ဇာ လက်ဗွေနှင့် User-Agent ကြည့်ရှုမှု

ဘရောက်ဆာတိုင်းသည် သင့် User-Agent၊ ပတ္တူနှင့် WebGL တင်ဆက်မှု၊ ဖောင့်များ၊ ဘာသာစကား၊ အချိန်ဇုန်နှင့် မျက်နှာပြင်အရွယ်အစား- သင့်အား မည်သည့်ကွတ်ကီးမှ မလိုအပ်ဘဲ အတူတကွ သိရှိနိုင်မည်ဖြစ်သည်။ ဤကိရိယာသည် သင့်ဘရောက်ဆာရှိ အဆိုပါအချက်ပြမှုများကို စက်တွင်းတွင်ဖတ်ပြီး ၎င်းတို့အား သင့်အား ပြသပေးသည်။ မည်သည့်အရာကိုမျှ အပ်လုဒ်မလုပ်ပါ။ စကင်န်သည် သင့်စက်ပေါ်တွင် လုံးလုံးလျားလျား အလုပ်လုပ်သည်။

သင်၏ဘရောက်ဆာ လက်ဗွေ၊ ယခုပင်

သင့်ဘရောက်ဆာမှ တိုက်ရိုက်ဖတ်ပါ။ တန်ဖိုးများသည် ဤစာမျက်နှာပေါ်တွင် ရှိနေသည် - မည်သည့်အရာကိုမျှ ဆာဗာသို့ ပေးပို့ခြင်းမပြုပါ။

ဘရောက်ဆာ လက်ဗွေဆိုတာ ဘာလဲ။

ဘရောင်ဇာလက်ဗွေသည် မည်သည့်ဝဘ်ဆိုက်ကိုမဆို သင့်ဘရောင်ဇာမှ ထိတွေ့နိုင်သော သတ္တိများမှ တည်ဆောက်ထားသော ပရိုဖိုင်တစ်ခုဖြစ်သည်- User-Agent စာကြောင်း၊ ထည့်သွင်းထားသော ဖောင့်များ၊ မျက်နှာပြင်ကြည်လင်ပြတ်သားမှု၊ အချိန်ဇုန်၊ ဘာသာစကားစာရင်းနှင့် သင့်စက်သည် ကင်းဗတ်စ်နှင့် WebGL မှတစ်ဆင့် ဂရပ်ဖစ်ကို မည်သို့ပြန်ဆိုမည်နည်း။ ပေါင်းစပ်မှုသည် မကြာခဏ ထူးခြားသောကြောင့်၊ သင်သည် ကွက်ကီးများကို ပိတ်ဆို့ခြင်း သို့မဟုတ် သီးသန့်မုဒ်ကို အသုံးပြုပါက နောက်ပိုင်းတွင် သင့်အား ဆိုက်တစ်ခုမှ မှတ်မိနိုင်မည်ဖြစ်သည်။

ကင်းဗတ်စ်နှင့် WebGL လက်ဗွေရာ အလုပ်လုပ်ပုံ

စာမျက်နှာတစ်ခုသည် ဝှက်ထားသော စာသားနှင့် ပုံသဏ္ဍာန်များကို ကင်းဗတ်စ်တစ်ခုသို့ ဆွဲယူပြီး pixels များကို ပြန်ဖတ်သည့်အခါ၊ သင်၏ GPU၊ ဂရပ်ဖစ်ဒရိုက်ဗာနှင့် ဖောင့်ဖော်ခြင်းရှိ သေးငယ်သော ကွဲပြားမှုများသည် သင့်စက်အတွက် တိကျသည့် တည်ငြိမ်သောပုံရိပ် hash ကို ဖြစ်ပေါ်စေပါသည်။ WebGL လက်ဗွေရာသည် ဖုံးကွယ်ထားသော GPU ရောင်းချသူနှင့် သရုပ်ဖော်သူကို ထိုနည်းအတိုင်း ဖတ်သည်။ ဤအချက်နှစ်ခုသည် cookie အများစုကိုရှင်းလင်းခြင်းနှင့် ဆက်ရှင်များတစ်လျှောက်တွင် ဆက်လက်ရှင်သန်နေသောကြောင့် ၎င်းတို့သည် အားကောင်းပါသည်။

မည်သည့်အချက်များ အများဆုံးဖော်ပြသနည်း။

သုတေသီများသည် entropy bits တွင် ပါဝါကို ခွဲခြားသတ်မှတ်ခြင်းကို တိုင်းတာသည်- ပိုများသော bits သည် သင့်တန်ဖိုးကို မျှဝေသူနည်းပါးလာသည်ဟု ဆိုလိုသည်။ ထုတ်ဝေသည့်လေ့လာမှုများက အားအကောင်းဆုံးပံ့ပိုးသူများထဲမှ ဖောင့်များ၊ User-Agent၊ ကင်းဗတ်နှင့် WebGL သရုပ်ဖော်မှုတို့နှင့်အတူ လက်ဗွေရာ အပြည့်အစုံကို 18 bits ဝန်းကျင်တွင် ထည့်သွင်းထားသည်။ အောက်ဖော်ပြပါဇယားသည် ပင်မရင်းမြစ်များ၏ ဆွေမျိုးအလေးချိန်ကို ပြသည်။

ဘရောင်ဇာ လက်ဗွေရာ အခင်းအကျင်း (2026 ခန့်မှန်းချက်)

အောက်တွင်ဖော်ပြထားသော ကိန်းဂဏန်းများသည် ထုတ်ဝေထားသော သုတေသနအခြေခံအချက်များမှ တည်ဆောက်ထားသော 2026 ခန့်မှန်းချက် (EFF Panopticlick 2010၊ AmIUnique 2016၊ လူစုလူဝေး 2018) နှင့် အချိန်ပေါင်းထည့်ခြင်း - ကိုးကားရန်အတွက်သာဖြစ်ပြီး၊ ပထမဆုံးလက်ဖြင့်တိုင်းတာခြင်းမဟုတ်ပါ။

အချက်ပြတန်ဖိုး
ထူးခြားသော ဒက်စ်တော့လက်ဗွေများ~83.6% (Panopticlick) မှ ~89.4% (AmIUnique) အခြေခံလိုင်း၊ Anti-fingerprinting ကိုပိတ်ထားသည့်အခါ 2026 တွင် 80% ကျော်ကျန်သေးသည်။
ထူးခြားသော မိုဘိုင်းလက်ဗွေရာများ~18.5% (Crowd in the Hiding 2018) - ဖုန်းများသည် တစ်သားတည်းဖြစ်နေသောကြောင့်၊
စုစုပေါင်း လက်ဗွေအင်ထရိုပီ~18.1 bits အခြေခံလိုင်း၊ အကြမ်းဖျင်းအားဖြင့် ဘရောက်ဆာ 286,000 တွင် 1 ခု
Canvas signal တစ်ခုတည်းထည့်ထားသော entropy ၏ ~ 5.7 bits (EFF)
အပြင်းထန်ဆုံး အချက်ပြမှုများဖောင့်အစုံ (34% ခန့်ကို ခွဲခြားသိမြင်နိုင်သော)၊ User-Agent၊ ပတ္တူ၊ WebGL သရုပ်ဖော်မှု၊ မျက်နှာပြင်
ဘရောက်ဆာ တိုးချဲ့မှုများအသုံးပြုသူများ၏ 56.86% ခန့်တွင် ထူးခြားသော extension set တစ်ခုရှိသည်။
လက်ဗွေရာ ဆန့်ကျင်ရေး အကျိုးသက်ရောက်မှုTor Browser၊ Brave နှင့် Safari ITP သည် အချက်ပြမှုများကို ပုံမှန်ဖြစ်စေခြင်း သို့မဟုတ် ပိတ်ဆို့ခြင်းဖြင့် ထူးခြားမှုကို သိသိသာသာ ဖြတ်တောက်နိုင်သည်

ခန့်မှန်းခြေ အခြေခံ- အများသူငှာ သုတေသနအခြေခံများ (EFF / AmIUnique / Hiding in the Crowd) နှင့် အချိန်ကို ပေါင်းထည့်ခြင်း၊ ဘရောက်ဆာများနှင့် လက်ဗွေရာဆန့်ကျင်ရေး ဆင့်ကဲပြောင်းလဲလာသည်နှင့်အမျှ ပထမဆုံးလက်ဖြင့် တိုင်းတာခြင်းနှင့် ပြောင်းလဲခြင်းမဟုတ်ပါ။

FAQ

ကျွန်ုပ်၏ဒေတာကို မည်သည့်နေရာတွင်မဆို ပေးပို့နိုင်ပါသလား။

နံပါတ်၊ စကင်န်သည် စံဝဘ် API များကို အသုံးပြု၍ သင့်ဘရောက်ဆာတွင် လုံးလုံးလျားလျား လုပ်ဆောင်ပါသည်။ တန်ဖိုးများကို စာမျက်နှာပေါ်တွင် ပြသထားပြီး မည်သည့်အခါမျှ မတင်ပါ။

User-Agent စာကြောင်းဆိုတာဘာလဲ။

၎င်းသည် ဘရောက်ဆာ၊ ၎င်း၏ဗားရှင်းနှင့် လည်ပတ်မှုစနစ်ကို အမည်ပေးသည့် တောင်းဆိုမှုတိုင်းဖြင့် သင့်ဘရောက်ဆာမှ ပေးပို့သည့် လိုင်းတစ်ခုဖြစ်သည်။ ၎င်းသည် ရှေးအကျဆုံးနှင့် အခြေခံအကျဆုံး လက်ဗွေအချက်ပြများထဲမှ တစ်ခုဖြစ်သည်။

ပတ္တူ hash သည် အဘယ်ကြောင့် အရေးကြီးသနည်း။

ကင်းဗတ်ပုံတစ်ပုံကို ဆွဲခြင်းနှင့် ပြန်ဖတ်ခြင်းသည် သေးငယ်သော GPU နှင့် ဖောင့်ပုံဖေါ်ခြင်းဆိုင်ရာ ကွဲပြားမှုများကို ထင်ရှားစေပြီး၊ သင်၏စက်ပစ္စည်းအတွက် မကြာခဏထူးခြားပြီး ဆက်ရှင်များတစ်လျှောက် တည်ငြိမ်သော hash တစ်ခုကို ထုတ်လုပ်ပေးပါသည်။

ကွတ်ကီးများကို ရှင်းလင်းခြင်းသည် ကျွန်ုပ်၏ လက်ဗွေကို ဝှက်ထားပါသလား။

နံပါတ်။ လက်ဗွေကို စက်နှင့် ဘရောင်ဇာ၏ ဂုဏ်သတ္တိများမှ တည်ဆောက်ထားခြင်းဖြစ်ပြီး သိမ်းဆည်းထားသော ကွတ်ကီးများမဟုတ်သောကြောင့် သင်ကွတ်ကီးများကို ရှင်းပစ်ပြီးနောက် သို့မဟုတ် သီးသန့်မုဒ်ကို အသုံးပြုပြီးနောက် ၎င်းသည် ဆက်ရှိနေနိုင်သည်။

ကျွန်ုပ်၏လက်ဗွေကို မည်သို့လျှော့ချနိုင်မည်နည်း။

အချက်ပြမှုများကို ပုံမှန်ဖြစ်စေသော (ဥပမာ Tor ဘရောက်ဆာ သို့မဟုတ် Brave)၊ သင့်ဘရောက်ဆာကို အပ်ဒိတ်လုပ်ထားပါ၊ ရှားပါးသော ဖောင့်များ၊ တိုးချဲ့မှုများနှင့် မျက်နှာပြင်ဆက်တင်များကို ရှောင်ကြဉ်ပါ။

ဤတန်ဖိုးများသည် မှန်ကန်ပါသလား။

၎င်းတို့ကို သင့်ဘရောက်ဆာမှ တိုက်ရိုက်ဖတ်ရှုနိုင်သောကြောင့် ၎င်းတို့သည် မည်သည့်ဝဘ်ဆိုဒ်ကိုမဆို မြင်နိုင်သည်ကို ထင်ဟပ်စေသည်။ အချို့သောဂုဏ်သတ္တိများ (deviceMemory, vendor, platform) ကို ရပ်ဆိုင်းထားသည် သို့မဟုတ် အနီးစပ်ဆုံးဖြစ်ပြီး ပျောက်ဆုံးနေနိုင်ပါသည်။

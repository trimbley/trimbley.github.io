ブラウザ指紋 / User-Agent 確認ツール

どのブラウザも、User-Agent、canvas と WebGL の描画、フォント、言語、タイムゾーン、画面サイズといった信号をサイトに渡しています。これらの組み合わせは、cookie がなくてもあなたを識別できることが多いのです。本ツールはこれらの信号をブラウザ内でローカルに読み取って表示します。データは一切アップロードされず、スキャンはすべてあなたの端末上で完結します。

今のあなたのブラウザ指紋

あなたのブラウザからリアルタイムに読み取ります。値はこのページに表示されるだけで、サーバーには送信されません。

ブラウザ指紋とは？

ブラウザ指紋とは、ブラウザが外部に公開する属性から作られるプロフィールです：User-Agent 文字列、インストール済みフォント、画面解像度、タイムゾーン、言語リスト、そして canvas や WebGL でグラフィックを描画する方法。その組み合わせはしばしば一意なので、cookie を遮断したりプライベートモードを使っても、再訪時にサイトはあなたを認識できます。

canvas と WebGL 指紋の仕組み

ページが隠した文字や図形を canvas に描画してピクセルを読み戻すと、GPU・グラフィックドライバ・フォント描画のわずかな違いが、あなたの端末に固有で安定した画像ハッシュを生みます。WebGL 指紋も同様にマスクされていない GPU のベンダーとレンダラーを読み取ります。この 2 つはほとんどの cookie 削除を生き延び、セッションをまたいで持続するため強力です。

最も多くを明かす信号

研究者は識別力を「エントロピーのビット数」で測ります。ビットが多いほど、あなたと同じ値の人は少なくなります。公開研究では完全な指紋を約 18 ビットと見積もり、フォント・User-Agent・canvas・WebGL レンダラーが最も強い寄与項です。下図は主な源の相対的な重みを示します。

ブラウザ指紋の現状（2026 推計）

以下は公開研究を基線（EFF Panopticlick 2010、AmIUnique 2016、Hiding in the Crowd 2018）に時間外挿した 2026 年の推計です——一次実測ではなく、参考用です。

信号値
デスクトップ指紋が一意である割合基線で約 83.6%（Panopticlick）〜89.4%（AmIUnique）。反指紋を切ると 2026 でも 80% 超
モバイル指紋が一意である割合約 18.5%（Hiding in the Crowd 2018）——端末が均質なため低い
完全な指紋の総エントロピー基線で約 18.1 ビット、ブラウザ約 28.6 万台に 1 台に相当
canvas 単独約 5.7 ビットのエントロピーを追加（EFF）
最も強い単一信号フォント集合（約 34% 識別可能）、User-Agent、canvas、WebGL レンダラー、画面
ブラウザ拡張機能約 56.86% のユーザーが一意な拡張機能の組み合わせを持つ
反指紋の効果Tor ブラウザ・Brave・Safari ITP は信号の均質化や遮断で一意性を大幅に下げられる

推計の根拠：公開研究の基線（EFF / AmIUnique / Hiding in the Crowd）＋時間外挿。一次実測ではなく、ブラウザと反指紋技術の進化で変わります。

FAQ

私のデータはどこかに送信されますか？

いいえ。スキャンは標準の Web API を使ってブラウザ内で完結し、値はページに表示されるだけでアップロードされません。

User-Agent 文字列とは？

ブラウザがすべてのリクエストで送る一行のテキストで、ブラウザ名・バージョン・OS を示します。最も古く基本的な指紋信号の 1 つです。

canvas ハッシュはなぜ重要？

canvas に画像を描いて読み戻すと、GPU とフォント描画の微差が露わになり、端末に固有でセッション間で安定したハッシュが生まれます。

cookie を消せば指紋を隠せますか？

いいえ。指紋は保存された cookie ではなく端末とブラウザの属性から作られるため、cookie 削除やプライベートモードの後も残ります。

指紋を減らすには？

信号を均質化するブラウザ（Tor ブラウザや Brave など）を使い、ブラウザを最新に保ち、目立つ稀なフォント・拡張機能・画面設定を避けましょう。

これらの値は正確？

あなたのブラウザから直接読み取るので、どのサイトにも見える内容そのものです。一部の属性（deviceMemory、vendor、platform）は非推奨または近似で、欠けることがあります。
